## Description

Odin Recipes is a static website that features three mouth-watering recipes. Each recipe is presented on a separate page with detailed instructions and ingredients.

## Files

- [index.html](./index.html): The main HTML file that serves as the entry point to the website.

- [recipes/meatballs.html](./recipes/meatballs.html): Page for Rao's Meatballs recipe.

- [recipes/salad.html](./recipes/salad.html): Page for Nancy's Chopped Salad recipe.

- [recipes/chocolate.html](./recipes/chocolate.html): Page for Chocolate Nemesis recipe.
